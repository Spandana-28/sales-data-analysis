# 📊 Sales Data Analysis

This project analyzes monthly sales data to identify top products, trends, and region-wise performance.

## 🔧 Tech Used
- Python
- Pandas
- Matplotlib
- GitHub

## 📁 Structure
- `data/`: contains the raw CSV sales data
- `notebooks/`: main analysis notebook
- `outputs/`: graphs and insights
